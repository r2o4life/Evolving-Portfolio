The Vertical Ads: Autonomous UI Paradigm project is a conceptual demonstration of an "Orchestration Layer" that automates interaction design.

In short, it proposes a paradigm shift from "Layouts to Loops": instead of designers manually researching friction and pushing pixels, they oversee AI-driven systems (Compile Engines) that detect user friction through real-time telemetry and automatically generate UI solutions.

The project is structured around four mutually exclusive "Compile Engines" that act as self-healing cybernetic loops:

Consumer Intent Loop: Ingests user telemetry (e.g., rapid drop-offs on flight filters) and automatically branches Figma files with alternative, production-ready layouts.
Advertiser Workflow Loop: Detects cognitive overload in Google Ads creation tools and outputs dynamic micro-copy and progressive disclosure wireframes.
Token & Component Loop: Identifies cross-surface visual regressions (e.g., broken mobile padding) and automatically generates GitHub PRs to patch the design system tokens.
Spec & Velocity Loop: Analyzes asynchronous design-to-engineering handoffs (e.g., PST to Bangalore) to automatically flag ambiguous specs and generate Jira tickets for missing edge cases.
Ultimately, it showcases a future where Interaction Designers act as system orchestrators, managing the constraints and inputs of an AI layer that actively repairs the UI in real-time.
