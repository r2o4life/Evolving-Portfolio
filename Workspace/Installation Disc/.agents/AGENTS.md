# Version Increment Rule

Every single time there is a new LLM chat response in this thread, you MUST upgrade the version number in `index.html`.
- The versioning starts from 1.00 and goes up to 50.00.
- Leverage smaller decimals (e.g., 1.01, 1.02, 1.03) to increment the version with each response.
- Make sure to update the version number within the `<span class="badge">v...</span>` element in `index.html` on every response you make in this project.
