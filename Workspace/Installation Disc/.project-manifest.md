# Project Manifest: AI Architect OS Installer

**Intent Seed for Architecture Sync Engine**

*   **Project Title:** AI Architect OS Installation Disc
*   **Target ID:** `architect-os-installer`
*   **Value Proposition:** A stylized web interface for deploying the Architect OS system prompt to bootstrap AI coding agents into a structured engineering workflow.
*   **Summary:** This project provides a frontend "installation disc" experience for users to copy an overriding system prompt. The prompt forces an LLM to utilize a `.blink/` configuration directory for PRD and stack specifications before writing code. It features a cyber-aesthetic, glassmorphic UI.
*   **GEMSG Alignment:** Governance (Enforcing AI coding standards), Growth (Scaling structured software output).

## A.C.T. Intent Hints
*   **Aesthetics:** Cyberpunk/glassmorphism vibe, using dark themes and ambient glows to represent an OS terminal environment.
*   **Clarity:** Clear, numbered onboarding steps (`01` through `04`) explaining the operational lifecycle of the prompt.
*   **Tone:** Authoritative, structured, and system-level.
